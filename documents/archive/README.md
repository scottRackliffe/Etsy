# Archive — superseded / retired documents

These documents are **retired and kept for historical reference only**. They describe the
**Listing Coach** and the four-mode / Listing-Workshop listing flow, all of which were **removed**
by **ADR-085** (unified listing lifecycle). Do **not** implement from them.

The current listing system is the single lifecycle on the inventory detail (ADR-081/082/085):
**Evaluate Data → Generate Listing → Evaluate Listing Quality → Publish to Etsy**, with new items
created via the inline SEMS editor (ADR-079).
